FAQ

What do I need to know before taking this course?

This course is meant to be an introduction to data science, so no prior knowledge in data science is required. However, you should be familiar with basic programming (variables, loops, functions) in order to understand the materials easily. Knowing Python syntax is highly recommended. (Recommended prerequisite/corequisite: CS 1110)

How heavy is the overall workload?

This is a project-based course, so you will have three big projects throughout the semester (and a few minor quiz assignments). The course is available as a 1-credit S/U class - it can also be taken for audit.

What will I be learning?

You will learn the basics of data science (how to read, write, and clean data), and the basics of machine learning (regressions and classifiers). We also offer separate workshops for neural networks and servers/Hadoop, so please check that as well.
